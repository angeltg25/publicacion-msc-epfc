Task: Crear una copia de datos de produccion
Tarea en donde se crea una copia de datos de produccion.
Purpose
Crear una copia de datos de produccion.
Relationships
RolesPrimary Performer:
  • Desarrollador Back End
Additional Performers:
InputsMandatory:
  • Doc de Plan de despliegue
  • informe de escenarios de prueba
  • informe de posibles soluciones
  • informe de pruebas unitaria con 15 casos de prueba de integracion
  • informe de pruebas unitaria con 15 casos de prueba de rendimiento
  • informe de pruebas unitaria con 15 casos de prueba de seguridad
    		Optional:
    • None
    Outputs
    • Doc de implementacion del entorno de prueba
      Main Description
      Si es posible y seguro, obtén una copia de datos de producción para utilizarla en el entorno de prueba. Esto permitirá realizar pruebas más realistas y obtener resultados más precisos. Asegúrate de proteger adecuadamente los datos y cumplir con los requisitos de privacidad y seguridad.
