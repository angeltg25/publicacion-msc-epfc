Task: Definir los objetivos de interaccion
Tarea donde se determina y define los objetivos de interaccion
Purpose
Definir y describir detalladamente los objetivos de interaccion
Relationships
RolesPrimary Performer:
  • Diseñador UI UX
Additional Performers:
InputsMandatory:
  • Doc Especificaciones de casos de uso de sistema
  • Doc prototipo de usuario
  • Doc Requerimientos funcionales
    		Optional:
    • None
    Outputs
    • Doc modelo de interaccion
      Main Description
      Antes de determinar el modelo de interacción, es importante comprender el contexto en el que se utilizará el sistema y conocer a los usuarios que interactuarán con él. Considera factores como las características demográficas de los usuarios, sus habilidades y conocimientos técnicos, y los objetivos que desean lograr a través del sistema.
