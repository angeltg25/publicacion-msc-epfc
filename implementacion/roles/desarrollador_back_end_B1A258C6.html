Role: Desarrollador Back End
Rol el cual tiene por funcion principal la implementacion de modelos de software y desarrollarlo a nivel de codigo.
Relationships
Primary Performs
  • Configurar el archivo de configuraciones en el codigo
  • configurar los controladores y las bibliotecas necesarias
  • Configurar los parametros de conexion de la base de datos
  • Crear modulos o paquetes
  • Establecer dependencias
  • Establecer la conexion
  • Implementar la estructura de directorios
  • Implementar los requerimientos con prioridad Alta
  • Implementar los requerimientos con prioridad Baja
  • Implementar los requerimientos con prioridad Media
  • Implementar relaciones entre entidades
Modifies
  • Doc de desarrollo de requerimientos
  • Doc implementacion y conexion de bases de datos
  • Doc informe de implementacion de modelo
    Main Description
    Este rol tiene la responsabilidad de implementar los modelos del software a nivel de codigo, desarrollar software, implementar bases de datos, desarrollar y mantener la seguridad del software, y entre otras funciones.
